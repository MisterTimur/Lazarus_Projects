# Lazarus_Projects

# Mini 3D editor for indie games <br/>

Формат файла <br/>
//---------------------------------------------------------- <br/>



O(ИМЯ) <br/>
  Создает обьект с заданым именем если его нету <br/>
  Eсли такой обьект есть то назначаю текущим <br/>
  и все последующие команды будут работать внутри него <br/>
  назначаю текщим контекстом <br/><br/>


O(self) сделать текущий обьект контекстом<br/><br/>  

K() далее следует анимация этот перфикс еще не доделан <br/><br/>  

 
//---------------------------------------------------------- <br/>
E(Имя) <br/>
  Создает Элемент в текущем контексте (Обьект,Элемент) <br/>
  если его нету если он есть то назначаю текщим контекстом <br/><br/>
//---------------------------------------------------------- <br/>
V(Имя) <br/> 
  Создает вершину в текущем контексте (Обьект,Элемент) <br/> 
  если его нету если он есть то назначаю текщим контекстом <br/><br/>
//---------------------------------------------------------- <br/>
L(Имя) <br/>
  Создает Линии в ткущем контексте (Обьект,Элемент) <br/>
  При создании в качестве вершин все 2 указываються <br/>
  текущий контекст (ОБьект,Элемент) его центр <br/><br/> 

d(Имя Вершины) <br/>
  Указвает вершину 1 в последней линии от команды L <br/>
e(Имя Вершины) <br/>
  Указвает вершину 2 в последней линии от команды L <br/><br/>
//---------------------------------------------------------- <br/>
P(Имя) <br/>
  Создает плоскость в ткущем контексте (Обьект,Элемент) <br/>
  При создании в качестве вершин все 4 указываються <br/>
  текущий контекст (ОБьект,Элемент) его центр  <br/><br/>
 
a(Имя Вершины) <br/>
  Указвает вершину 1 в последней плоскости от команды P <br/>
b(Имя Вершины) <br/> 
  Указвает вершину 2 в последней плоскости от команды P <br/>
c(Имя Вершины) <br/>
  Указвает вершину 3 в последней плоскости от команды P <br/>
d(Имя Вершины) <br/>
  Указвает вершину 4 в последней плоскости от команды P <br/>

//---------------------------------------------------------- <br/>
<br/>
  Большие x y z <br/>
<br/>
X(Значение float)  <br/>
  Записывает в координату X текщего примитива <br/>
Y(Значение float)  <br/>
  Записывает в координату Y текщего примитива <br/>
Z(Значение float) <br/> 
  Записывает в координату Z текщего примитива <br/>
<br/>

//---------------------------------------------------------- <br/>
<br/> 
  Маленькие x y z <br/>
 <br/> 
x(Значение float)  <br/>
  Записывает в Углы наклона y текщего примитива <br/>
y(Значение float) <br/> 
  Записывает в Углы наклона y текщего примитива <br/>
z(Значение float) <br/> 
  Записывает в Углы наклона z текщего примитива <br/>

<br/>
//---------------------------------------------------------- <br/>
<br/>
 
  Больше  R G B A Цвета примитива <br/>
<br/>  
R(Значение 0..255) <br/> 
  Записывает Краный цвет составляющая текущего примитива <br/>
G(Значение 0..255) <br/> 
  Записывает Зеленеый цвет составляющая текущего примитива <br/>
B(Значение 0..255) <br/> 
  Записывает Голубой цвет составляющая текущего примитива <br/>
A(Значение 0..255) <br/> 
  Записывает Прозрачность составляющая текущего примитива <br/>
